3. Documented vs. Unverified Distinction

Critical Principle We've Established:

  • Documented cases have physical evidence, court convictions, victim testimony (high credibility)
  • Unverified claims lack verifiable evidence, rely on anonymous sources or speculation (low credibility)
  • Some conspiracy theories contain kernels of truth embedded in speculation (e.g., Epstein files contain documented network)
  • Most conspiracy theories are entirely speculative or based on misinterpretation

Application to Future Research:

  • When investigating unverified claims, explicitly state "this lacks verifiable evidence"
  • When investigating documented cases, emphasize physical evidence, court records, victim testimony
  • Distinguish between "documented pattern" vs. "isolated incident"
  • Never assume conspiracy theory is true without evidence, but never dismiss it without investigation

4. Source Diversity & Bias Mitigation

What We've Done Well:

  • Presenting multiple sources across spectrum (mainstream, academic, alternative, fringe)
  • Using credibility ratings to signal evidence quality
  • Including conspiracy theory perspectives alongside academic rebuttals
  • Distinguishing between documented evidence vs. speculative claims

What We Can Improve:

  • Seek out primary sources for conspiracy claims (whenever possible)
  • Identify patterns of source suppression (not just reporting what they say)
  • Document funding sources and institutional ties for all sources
  • Cross-reference claims across multiple independent sources when possible
